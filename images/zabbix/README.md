# zabbix Docker Image

Docker image for zabbix.

## Usage

```bash
# Build the image
docker build -t zabbix .

# Run the container
docker run -d zabbix
```

## Files

- `Dockerfile`: Main Docker image definition
- `build.sh`: Build script for image setup
