# webcron Docker Image

Docker image for webcron.

## Usage

```bash
# Build the image
docker build -t webcron .

# Run the container
docker run -d webcron
```

## Files

- `Dockerfile`: Main Docker image definition
- `build.sh`: Build script for image setup
- `entry.sh`: Container entrypoint script
