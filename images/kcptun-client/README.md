# kcptun-client Docker Image

Docker image for kcptun-client.

## Usage

```bash
# Build the image
docker build -t kcptun-client .

# Run the container
docker run -d kcptun-client
```

## Files

- `Dockerfile`: Main Docker image definition

- `entry.sh`: Container entrypoint script
